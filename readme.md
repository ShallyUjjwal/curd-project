# CURD Project

A simple CRUD application built using Node.js, Express.js, MongoDB, and Mongoose.

This project demonstrates basic backend development concepts, routing, database operations, and server-side rendering.

## 🚀 Features

- Create data
- Read data
- Update data
- Delete data
- MongoDB integration

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
